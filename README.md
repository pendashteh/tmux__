# `tmux__`: an undies helper for tmux

```bash
[bash] /> tmux__ 

Undies app name: tmux__
Undies API version: 4

Parameters:
n/a

Available tasks:
 - help      # @arg task @does declare task implementation
 - mouse     # @arg on|off @does triggers mouse control
 - sync      # @arg on|off @does sync all panes on the window.

Usage:
 $ tmux__ [__param=value] [task]
```
